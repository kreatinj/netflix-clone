# netflix-clone

넷플릭스 클론 프로젝트

## 기능요구사항

- 헤더 (상단고정)
- 히어로 (상단 배너: 배경 이미지 또는 정지 포스터 + 오버레이 + 제목/설명/버튼)
- 카테고리 섹션 3개 이상(가로 카드 행, 각 행 6개 카드)
- 푸터

## 프로그래밍 요구사항

### HTML 구현

- 시맨틱 태그 사용
  - 예) `<header>` `<main>` `<section>` `<footer>` ...
  - 태그 사용시 의미(의도)를 잘 담는 것을 사용한다.
  - div, span을 금지하는것은 아님.
- 이미지태그는 alt 속성을 추가해서 설명한다.(웹접근성)
- 링크는 형식만 표현(실제 이동/재생 없음)
  - 카드/버튼은 `<a href="#">` 또는 `<button disabled>` 등으로 UI 형태만 만든다

### 크기/간격 규칙

- 고정 너비 페이지: 예) 1200~1440px 중 직접 선택
  - .page { width: 1440px; margin: 0 auto; }
  - 반응형웹을 만들지 않는다.(금지는 아니지만 비추천)
- 헤더 높이: 자유 (예: 72px)
- 카드 비율: 2:3 또는 16:9 중 선택
- 여백/간격은 8px/16px/24px/32px 단위 사용 (일관성 유지)

### 재사용 요소

- 디자인 토큰(CSS 변수) 사용
  - 색상/간격/반경/그림자/폰트크기 중 일부를 :root에 변수로 선언 후 재사용해본다.

### 레이아웃 방식

- Flexbox 방식, CSS-Grid 방식 중에 최소 한가지 이상을 사용한다.
- 예) 바깥쪽: Flexbox 사용, 카드가로 배치 : CSS-Grid 사용

### CSS 기반 동적 효과

- CSS 만으로 구현해보기 : 카드 호버 시 확대 (scale up)

### 기본적인 접근성을 고려해 구현

- 시멘틱 태그 활용으로 구조와 의미를 명확히 한다.
- 이미지에는 대체 텍스트(alt) 를 제공한다.
- Tab 키를 활용한 키보드 탐색이 가능하게 하고, 포커스 상태가 보이도록 한다.
- 색상 대비를 확인해, 글자와 배경이 충분히 구분되도록 한다.
