# netflix-clone

넷플릭스 클론 프로젝트

## 기능요구사항

### 알림 모달

- 헤더 영역 `알림`에 마우스를 호버했을 때, 알림 모달이 노출된다.
- 알림 모달이 열리고 닫힐 때, 페이드 인/아웃(fade in/out) 애니메이션 효과가 적용된다.
- 모달 내용물에 마우스 호버 시, 배경색이 변경된다.
- 스크롤 기능이 적용된다.

### 프로필 모달

- 헤더 영역 `프로필`에 마우스를 호버했을 때, 프로필 모달이 노출된다.
- 프로필 모달이 열리고 닫힐 때, 페이드 인/아웃(fade in/out) 애니메이션 효과가 적용된다.
- 모달이 열리고 닫힐 때, 화살표 아이콘이 회전 애니메이션 효과가 적용된다.
  - 열릴 때, 시계방향 180도 회전
  - 닫힐 때, 반시계방향 180도 회전
- 모달 내용물에 마우스 호버 시, 밑줄이 표시된다.

### 카드형 콘텐츠 슬라이딩 기능

1. 카드형 콘텐츠 슬라이딩 (Carousel / Slider)

- 카드들은 가로 방향으로 나열되며, 좌/우 네이게이션 버튼(<, >)을 클릭하면 슬라이드된다.
- 네이게이터 버튼 호버 시, 버튼 배경색이 변경된다.
- 슬라이드 시 애니메이션 전환 효과가 적용된다.
- 화면 비율에 따라 한 번에 보여지는 카드 수가 다른데, 여기서는 6개로 고정한다.
- 슬라이드하면 6개씩 이동한다. 마지막 슬라이드에서는 남은 카드 수만큼만 이동한다.

2. 페이지네이션 (Page Indicator)

- 각 카테고리 블록의 우측 상단에는 현재 슬라이드 위치를 나타내는 페이지 인디케이터가 표시된다.
- 페이지네이션은 슬라이드 이동에 맞춰 업데이트된다.
